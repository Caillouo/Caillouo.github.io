Projet Solurisk
L'objectif de ce projet était de réaliser une refonte totale du site web d'un client.

Le site web initial était réalisé sous Wordpress et le client souhaitait améliorer son référencement et la beauté de son site web.
Nous avons donc réalisé une refonte totale du site web en utilisant le framework Laravel.

Nous avons développé un front-office pour les clients de Solurisk et un back-office pour les administrateurs du site web.
Le front-office est entièrement personnalisable par les administrateurs du site web par un simple clic sur les champs des pages
Le back-office permet de gérer les utilisateurs, les etablissements et le matériel de ceux-ci
Enfin, ce back-office permet de réaliser des rapports d'intervention avec un export PDF envoyé par mail.
Projet Temple Trouble sur Unity
L'objectif de ce projet était de se familiariser avec le développement de jeu 3D sur Unity.

Le jeu est un jeu de labyrinthe où le joueur doit trouver la sortie en évitant les obstacles.
Il possède un système d'objets à récupérer pour avancer dans le labyrinthe.
  • Des clés pour ouvrir la porte de fin. Le nombre nécessaire est paramétrable
  • Des marteaux pour pouvoir casser des murs intérieurs du labyrinthe
  • Des cailloux pour marquer son chemin, le nombre est paramétrable
  • Des planches pour combler les trous qui peuvent être rencontrés
    • La taille du labyrinthe est également paramétrable pour augmenter la difficulté du jeu.
    Projet Ressource Sharing en Java
    L'objectif de ce projet était de créer une plateforme de partage de ressource et savoir-faire

    Nous avons donc réalisé une application web en utilisant le framework SpringBoot.
    Cette application permet de partager des ressources et des savoir-faire entre les utilisateurs.
    Les utilisateurs peuvent créer des comptes et ensuite renseigner des ressources et des savoir-faire qu'ils possédent.
    Les utilisateurs peuvent ensuite consulter les ressources et les savoir-faire des autres utilisateurs.
    Nous avons également développé un système de messagerie entre utilisateurs pour faciliter les échanges.
    Et évidemment un système de demande d'emprunt entre utilisateurs avec un système de validation par l'utilisateur prêteur.
    Projet de création de trajet sur carte intéractive
    L'objectif de ce projet était de créer une carte intéractive pour des utilisateurs externes

    Cette carte devait donc permettre à des utilisateurs de créer des trajets entre différents points.
    Une fois le trajet créé par l'utilisateur, il pouvait générer un graphique de dénivelé du trajet.
    Une fois le graphique généré, il pouvait parcourir celui-ci pour voir les différentes altitudes du trajet.
    Malheureusement, ce projet ne fonctionne plus à l'heure actuelle car l'API utilisée n'est plus disponible.
    Projet de carte informative avec gestion de projet
    L'objectif de ce projet était de créer une carte à destination de nouveaux étudiants
    Le contexte était ensuite libre pour les fonctionnalités de la carte
    Une autre partie du projet était de réaliser une gestion de projet complète avec cahier des charges, discution avec clients...

    Nous avons donc décidé de choisir un projet de découverte des quartiers d'Amiens
    Avec une visualisation des quartiers, des points d'intérêts et des points de culture de chacuns de ces quartiers

    Projet de gestion de bar en .NET
    L'objectif de ce projet était de créer une application web à destination d'un bar

    Nous avons donc réalisé une application web en utilisant le framework .NET.
    Cette application permet de gérer les commandes des clients, les stocks, les tables et les employés.
    Il y avait 3 interfaces différentes :
    • Une interface pour les serveurs leur permettant de prendre les commandes et les servir
    • Une interface pour les barmans leur permettant de préparer les commandes
    • Une interface pour les gérants leur permettant de gérer les stocks, les tables et les employés

    Projet d'auto-évaluation sur Angular
    L'objectif de ce projet était de créer une plateforme web d'auto-évaluation à destination d'étudiants

    Nous avons donc réalisé une application web en utilisant le framework Angular.
    Les étudiants peuvent s'inscrire à des projets nécessitant des compétences spécifiques.
    Les étudiants peuvent ensuite s'auto-évaluer sur ces compétences pour voir leur progression.
    Les enseignants quant à eux peuvent créer des projets et des compétences pour les étudiants.
    Ils peuvent également voir les résultats des étudiants pour chaque compétence sur leurs projets.
